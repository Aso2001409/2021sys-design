```uml
@startuml

opt 会員情報未登録
ユーザー -> Webサーバー : ユーザー登録
Webサーバー -> DBサーバー : ユーザー登録
DBサーバー -> DBサーバー : 登録処理
DBサーバー -> Webサーバー : 登録結果

alt 成功
Webサーバー -> ユーザー : 成功メッセージ表示
else 失敗
Webサーバー -> ユーザー : 失敗メッセージ表示
end
end

@startuml
ユーザー -> Webサーバー : ログイン
Webサーバー -> DBサーバー : ログイン照会
DBサーバー -> DBサーバー : ログイン処理実行
DBサーバー -> Webサーバー : 処理結果

activate ユーザー
alt 認証成功
Webサーバー -> ユーザー : 商品一覧ページ表示
else 認証失敗
Webサーバー -> ユーザー : 失敗表示
end

opt ログイン中
ユーザー -> Webサーバー :ログアウト
Webサーバー -> DBサーバー :ログアウト照会
DBサーバー -> DBサーバー : ログアウト処理
DBサーバー -> Webサーバー : 処理結果
Webサーバー -> ユーザー : トップページ表示
end

deactivate ユーザー
@enduml
```
